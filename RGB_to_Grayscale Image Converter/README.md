## RGB Image to Grayscale Image Converter :

It is a Python program that can be used to convert images in RGB to a grayscale image. 

#### Built With
1) OpenCV

#### Prerequisites
1) pip install opencv-python
2) pip install numpy

#### Run the app
1) Open CLI
2) Run python RGB_to_Enhanced_Image.py
